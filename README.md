# 🎯 Готовая сборка YouTube Morphe — просто скачай и пользуйся

Эта инструкция поможет вам быстро настроить мод после установки.

## 📲 Пошаговая настройка

1. **Запустите YouTube** и выполните базовый вход в аккаунт (всё стандартно, как в официальном приложении).
2. **Перейдите в настройки** приложения (иконка профиля → **Настройки**).
3. **Зайдите в раздел Morphe** (внутри настроек YouTube).
4. Выберите **«Прочие»** → **«Резервное копирование настроек»** → **«Импорт из файла»**.
5. **Укажите файл** настроек, который приложен к релизу (например, `youtube_morphe_settings_*.txt`).
6. **Готово!** Все оптимальные настройки уже применены.
